## About Life Reminders

**Your Life Has Deadlines. We Remember Them.**

A smart, privacy-first deadline tracker for the entire Apple ecosystem. Never miss another expiry, renewal, or recurring obligation across 45+ life categories — from passports and tax deadlines to pet vaccines and oil changes.

### Available On
- 📱 iPhone
- 📲 iPad
- 💻 Mac (native, not Catalyst)
- ⌚ Apple Watch

### Key Features

- 🎤 **Voice-first input** — speak naturally, on-device transcription, no internet needed
- 📅 **45+ smart categories** — passport, visa, vehicle, home, tax, pet care, and more
- ⏰ **Smart grace periods** — passport alerts 1 year out, bills 3 days out
- 🔁 **Auto-recurrence** — set once, birthdays/bills/passports renew automatically
- 🔔 **Overdue nagging** — daily reminders until you mark done (configurable 1 week to 3 months)
- 👀 **Three views** — Dashboard (urgency), Timeline (horizon), Calendar (week/month/year)
- 👨‍👩‍👧‍👦 **Family tagging** — track whose passport, whose birthday, whose appointment
- ☁️ **iCloud sync** — Pro users sync across all devices via private CloudKit
- 🔒 **100% private** — no accounts, no telemetry, no tracking, no server
